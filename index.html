Welcome to the Computer Science Syllabus! Explore the topics and enhance your knowledge.

Computer System

Learn about the components of a computer system, including input/output devices, memory, CPU, and storage devices.

Number System

Understand different number systems like binary, octal, decimal, and hexadecimal, and their conversions.

Software

Explore system software and application software, including operating systems and utility programs.

Application Package

Get hands-on experience with application packages like MS Word, Excel, and PowerPoint.

Programming

Learn programming concepts and logic using the C language.

Web Technology

Discover the basics of web development using HTML and CSS.

Multimedia

Understand the components and applications of multimedia in various fields.

Cyber Law

Learn about information security, cyber ethics, and legal aspects of using technology.
