Introduction to Internet

Internet is the global network of networks. or, It is huge collection of computers all over the world that is inter-connected to one another in various ways to form a web-net.

Internet services and applications

  1. World Wide Web (WWW): WWW is a series of services that are interconnected through hypertext. It provides links in the web document. It is the most important service provided by the Internet. It was created in 1990 at the European Particle Physics Laboratory in Geneva, Switzerland, to exchange and share data through the Internet using the protocol known as Hypertext Transfer Protocol (HTTP).
  2. Web Browser: A web browser is a computer program that accesses web pages and displays them on a computer screen. It is client software that allows users to display and interact with a web page. Web browsers help find, retrieve, view, send, and receive information over the Internet. Without a web browser, we cannot open the WWW. Some popular web browsers are Internet Explorer, Safari, Mozilla Firefox, Opera, Deepnet Explorer, Netscape Navigator, and Mosaic.
  3. FTP - File Transfer Protocol: It is an Internet tool that permits an Internet user to move or transfer a file from one computer to another. The files may contain data, graphics, text, etc.
  4. Electronic Mail (E-mail): Sending and receiving messages electronically through the Internet is called E-mail. It is the most widely used feature of the Internet. E-mail messages include digital text, graphics, video, animation, and audio data.
  5. Voicemail: Voicemail is a service offered by specialized computer hardware and software that answers telephone calls and records audio messages. We can talk to people using voicemail in an easy way.
  6. USENET – View and News: USENET, also called View and News, is the BBS (Bulletin Board Service) of the Internet. The messages in this BBS are organized into thousands of newsgroups, which cover specific areas of the Internet.
  7. Chat: Chat is a special feature provided by the Internet to communicate with others by typing in real time. A user can log into any chat room and connect with any person who is online. It is a very easy way to exchange ideas, feelings, and information. Offline messaging is also available if your friend is not online.
  8. IRC(Internet Relay Chat): Internet Relay Chat (IRC) is a huge multi-user live chat facility. There are a number of major IRC servers around the world, which are linked to each other. Anyone can create a channel and all others in the channel see anything that anyone types in a given channel.
  9. Video Conference: It is known as web cam chatting these days. This Video Conferencing uses the same technology as IRC but also provides sound and video pictures. It enables direct face-to-face communication across networks. It also provides real-time visual communications (online chat).
  10. Universal Resource Locator (URL): URL is a unique address that identifies its location on the Internet. Web browsers utilize the URL to retrieve a file from the computer on which it resides. The format of the URL is given below:

    • http - protocol
    • www - host computer name
    • animalplanet - domain name
    • com - domain type
  11. Navigators – Information Servers These are special features of Internet to discover, locate, and retrieve information on various objects. Some of the well-known utilities are Gopher, Archie, Hytelnet, WAIS, and WWW. The WWW navigator is the most popular and has resulted in the massive growth of Internet resources.
  12. TELNET – Remote Login: Telnet allows users to login to a host computer from a remote computer using a telnet program. In other words, it gives the user the opportunity to be on one computer system and work on another computer system, which may be across the street or thousands of miles away.
  13. E-Fax (Electronic Fax): It is another facility available in the Internet to send and receive faxes directly on our computer. It is very same of E-mail. The E-Fax message first goes to the Internet server, then the Internet server sends it to its destination whereas E-mail message goes to the local file server.

Terms related to the Internet

  1. ISP (Internet Service Provider): An institution that provides access to the Internet, usually for money. Examples in Nepal include Mercantile, Everest Net, and World Link.
  2. Link: A quick way to navigate to other web pages from your current page. Links are usually displayed in a different color than the surrounding text.
  3. Search Engine: Software that searches the web for sites. Examples include Yahoo, Excite, Lycos, and Alta Vista.
  4. Domain Name and IP Address: A unique name that identifies an Internet site. Domain names have two or more parts separated by dots. The IP address is the corresponding numeric identifier that facilitates data transfer between computers.
  5. DNS Server: A server that translates domain names into IP addresses. For example, Internet.net.
  6. Web Server: A site or program that hosts web pages and responds to requests from web browsers.
  7. Usenet Newsgroups or Net News: Specialized forums for discussing specific topics. For example, typing "Sci" for science or "Rec" for recreation in the search box will provide related information.
  8. Mailing Lists: Special email lists delivered to subscribers' mailboxes.
  9. POP Server: A server that stores incoming emails for users to collect. The Post Office Protocol (POP) allows single-user hosts to read mail from a server.
  10. SMTP Server: A server that transfers electronic mail between computers using the Simple Mail Transfer Protocol (SMTP).
  11. FTP (File Transfer Protocol): A protocol used to transfer files between computers on a network. Other protocols include TCP/IP, HTTP, and SMTP.
  12. Anonymous FTP: A facility to access archived files, such as software, documents, and media, without requiring user identification.
  13. Archived Files: A collection of files that are not publicly accessible.
  14. HTTP (Hypertext Transfer Protocol): A protocol used by web browsers and servers to communicate and transfer HTML documents.
  15. Proxy Server: A server that improves access speed and updates information on websites.
  16. Internet Roaming: A facility that allows users to access their Internet account from anywhere in the world, provided their ISP is part of the Global Roaming Community.
  17. Email Address Breakdown:
    • krishna - Username
    • @ - Separates the username and domain name
    • wlink - ISP name providing the internet access account
    • .com - Indicates a commercial organization or business
    • .np - Country code for Nepal

Introduction to HTML

HTML stands for Hyper Text Markup Language which are used to create web pages that are used in internet to exchange data and information. HTML file itself is a text file which contains several tags and attributes. HTML was develop by Tim Berner Lee that was based on SGML (Standard Generalized Markup Language). The tags and attributes used to create HTML file are displayed using web browser. Web browser helps to understand all the tags and attributes given by the user and generate the contain of the page. The different version of HTML are: HTML, HTML+, HTML 1.0, HTML2.0, HTML3.2, HTML 4.01 and now HTML 5.0.

Features of HTML

  1. It is used to create a basic layout or design of a web page.
  2. Without HTML, the worldwide web wouldn't exist.
  3. It allows features like embedding images, videos, audio, etc.
  4. It allows users to customize and format characters such as bold, italic, underline, etc.
  5. It helps to create links and lists.

Common Terminologies related to Web Designing

  1. Webpage: It is generally an HTML document or written in any other scripting language that helps to execute information in the form of text, images, or any other component through the internet.
  2. Website: The collection of interrelated webpages is known as a website. It may contain any piece of information or may have information that is interrelated with several webpages connected through hyperlinks. All websites have two parts:
    • Domain Name: The name used to access the website.
    • Web Server: It is a storage location where every piece of information associated with the website is stored.
  3. Home Page: The introductory page that is executed first when we access a website.
  4. Web Browser: It is an application program that helps to access any content of webpages, which may be in any form, using HTTP (Hypertext Transfer Protocol). This protocol helps to make communication between the web browser and the web server. Examples include Google Chrome, Internet Explorer (Microsoft Edge), Safari, Mozilla Firefox, and Netscape Navigator.
  5. URL: It stands for Uniform Resource Locator. It is a standard way of referencing a protocol, host, and directory.

Database Management System

Data are the raw facts and figures. Data are isolated, uninterpreted, and don't give any sense unless and until they are processed. Simply, they are the entities collected from the random world which undergo processing. For example, Ram, 19, 12, Pokhara.

Information

Information is the meaningful result obtained after processing data, which gives some meaning and sense. Generally, it is a processed version of data. For example, Ram is a student whose age is 19, and he lives in Pokhara-12.

Database

As we know, information is obtained after processing data. Hence, we need data processing to convert data into information. Therefore, the systematic and scientific arrangement and organization of data so that we can retrieve it whenever required quickly and easily is called a database. A database is an organized collection of data for converting data into information so that it can give some meaningful results. For example, Dictionary, Mark ledger, Census, Telephone directory, etc.

Database Management System (DBMS)

Since manual collection and organization of data are time-consuming, unsecured, and complex, there is an application program that helps to maintain a database, i.e., systematic and scientific organization and collection of data. This allows us to retrieve any piece of information within a fraction of time without any effort. Hence, the software or system built to fulfill the above requirements is known as a Database Management System (DBMS). In other words, it is a collection of software that manages data stored in a database. The main purpose of DBMS is to store data, process it, and obtain desired information. For example, MS-Access, MySQL, Oracle, FoxPro, dBase.

Advantages/Features/Importance of DBMS

  • It allows users to search quickly and easily.
  • It helps to reduce data redundancy (repetition).
  • It allows users to share data over a network.
  • It helps in data security and privacy.
  • It improves data inconsistency (variance).
  • It provides an organized way of storing data.

Disadvantages of DBMS

  • Installation and operating costs are higher.
  • It requires more disk space.
  • It may sometimes have security and privacy issues.
  • Technical manpower (DBA) is required to maintain and operate.

Practise Questions

1. What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages.

2. What is a tag in HTML?

A tag is a piece of markup language used to define elements in HTML. Tags are enclosed in angle brackets, like <tagname>.

3. What is the difference between HTML and HTML5?

HTML5 is the latest version of HTML. It introduces new elements, attributes, and APIs for better multimedia support, semantic elements, and improved performance compared to older versions of HTML.

4. What is the purpose of the <head> tag in HTML?

The <head> tag contains metadata about the HTML document, such as the title, character set, styles, scripts, and other information that is not displayed on the webpage itself.

5. What is the use of the <a> tag in HTML?

The <a> tag is used to create hyperlinks in HTML. It allows users to navigate to other web pages or resources by clicking on the link.

6. What is the purpose of the <img> tag in HTML?

The <img> tag is used to embed images in an HTML document. It requires the src attribute to specify the image file's path.

7. What is the difference between <div> and <span> in HTML?

The <div> tag is a block-level element used to group larger sections of content, while the <span> tag is an inline element used to style smaller portions of text or content.

8. What is the purpose of the <table> tag in HTML?

The <table> tag is used to create tables in HTML. It consists of rows (<tr>), headers (<th>), and data cells (<td>).

9. What is the use of the <form> tag in HTML?

The <form> tag is used to create forms in HTML. It allows users to input data, which can be submitted to a server for processing.

10. What is the purpose of the <meta> tag in HTML?

The <meta> tag is used to provide metadata about the HTML document, such as character encoding, author, description, and keywords for search engines.
